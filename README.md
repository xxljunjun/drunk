### 该项目是仿奈雪的电商类微信小程序

## 开发

```bash
# 克隆项目
git clone https://github.com/xxljunjun/drunk.git

# 进入项目目录
cd drunk

# 安装依赖
npm install

#启动服务
1、安装微信者开发工具
2、在h-builder上设置微信者开发工具的安装路径
3、在h-builder上设置端口号http://10.240.5.230:64579
4、在h-builder上运行到小程序
```
